# Asciify

Asciify is a Python library designed to provide functionalities for displaying instructions in a user-friendly manner. It allows users to output text to standard output or to a specified file descriptor, making it versatile for various applications.

## Features

- Display maze cells on standard output or a file.
- Easy integration with command-line interfaces.

## Usage

To use Asciify, you can import the necessary functions from the library. Below is a simple example of how to display instructions:

```python
from asciify.display import display_instructions

# Display instructions to standard output
display_instructions("Your instructions here.")
```
