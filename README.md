# Instructions
Our program runs with the following command:

```bash
python3 a_maze_ing.py config.txt
```

• `a_maze_ing.py` is our main program file.

• `config.txt` is our config file where we can insert the following : 

```bash
WIDTH=32
HEIGHT=16
ENTRY=1,1
EXIT=2,2
OUTPUT_FILE=maze.txt
PERFECT=True
```
